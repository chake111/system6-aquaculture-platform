# 产品方案书：智慧水产养殖监测与调控平台

## 1. 产品概述

面向广西鲈鱼养殖基地的水质监测与智能调控平台，覆盖水质实时监控、溶氧预警处置、密度分析、效益报表和运维管理。适配鱼塘潮湿环境，支持语音登录和弱网离线操作。

## 2. 用户角色与权限

| 角色 | 登录方式 | 核心权限 |
| --- | --- | --- |
| 养殖户 | 手机号 + 密码 / 语音 | 查看水质、确认执行建议、提交处置反馈 |
| 技术员 | 手机号 + 密码 / 语音 | 复核建议、审核密度分析、关闭预警、导出报表 |
| 管理员 | 手机号 + 密码 / 语音 | 运维监控、审计日志、阈值配置、归档审批 |

跨基地（base_id）数据隔离：用户只能访问其 pond_scope 授权范围内的塘口数据。

## 3. 功能需求

### FR-01 应用外壳与塘口登录
移动端适配登录页，支持高对比度模式、大按钮触控、在线/离线状态指示。支持语音输入（说角色名登录）和文字/按钮降级。

### FR-02 应用外壳与现场登录
登录后导航按角色过滤，未认证用户重定向到登录页，未授权菜单隐藏。

### FR-03 水质监测
授权用户可查看各塘口溶氧（DO）和 pH 最新值及历史趋势。读数标注来源和质量状态。阈值评估产生异常结果，供调控和预警模块消费。

### FR-04 密度分析
技术员注册声呐/图像样本，获取密度分析结果（含模型版本和误差范围）。结果需复核后才能发布放流/捕捞建议。养殖户可提交执行反馈。模拟样本和效果指标明确标注。

### FR-05 溶氧调控
低溶氧输入产生增氧建议（含原因和规则版本）。建议遵循状态机流转：复核 -> 确认 -> 执行 -> 评价。真实设备执行需授权；演示模式使用明确标注的模拟适配器。

### FR-06 预警中心
阈值异常创建或合并活跃告警。模拟通知渠道可返回送达/失败状态并重试。授权用户可确认、处置和关闭告警。时间线和审计追踪完整。

### FR-07 弱网同步
边缘节点上传不可变事件批次。离线缓存保留至少 7 天模拟数据。恢复后重复事件不产生重复读数/告警。UI 明确显示离线和最后同步状态。

### FR-08 数据台账与报表
管理员维护基地、塘口和设备主数据。技术员/管理员可查询指标定义并在授权范围内生成脱敏导出。所有指标声明数据来源、统计周期和模拟/真实属性。

### FR-09 效益指标与脱敏导出
跟踪五类效益指标（电费、成活率、亩产、饲料成本、综合成本）。导出脱敏处理：去除明文手机号、原始媒体存储地址和敏感审计内容。导出记录含用途、策略、到期时间和审计追踪。

### FR-10 运维健康与审计
管理员可查看服务/边缘/通知/同步健康状态和审计事件。阈值配置维护版本和变更审计。敏感配置和数据不在 API 响应或日志中泄露。

### FR-11 身份与权限基础
三种角色（养殖户/技术员/管理员）配合 base_id/pond_id 数据范围。用户范围强制仓库级查询过滤。离线授权时限（最长 7 天）并限制敏感操作。

### FR-12 归档与销毁（P2）
归档/销毁记录需审批引用、数据范围、操作类型、执行人、时间戳和证据引用。导出文件归档和审批销毁生成审计事件。演示阶段仅验证流程和证据，不验证真实监管集成。

## 4. 非功能需求

| 编号 | 需求 | 说明 |
| --- | --- | --- |
| NFR-01 | 请求追踪 | 所有 HTTP 响应包含 `X-Trace-Id` 头 |
| NFR-02 | 统一错误格式 | 所有 API 错误返回 `{"error": {"code": "...", "message": "..."}}` |
| NFR-03 | UTC 时间 | 所有持久化数据使用 UTC 时间 |
| NFR-04 | 来源标识 | 所有数据记录包含 `source_mode` 或 `execution_mode` 字段 |
| NFR-05 | 幂等性 | 批量上传、执行、导出和归档操作使用幂等键防止重复 |
| NFR-06 | 边缘同步韧性 | 断网后批量上传，重复事件幂等拒绝，离线缓存标注数据新鲜度 |
| NFR-07 | 角色访问控制 | 每个 API 端点强制角色和数据范围过滤 |
| NFR-08 | 审计日志 | 登录、授权、执行、处置、通知、同步拒绝、导出和配置变更写入 audit_logs |
| NFR-09 | 敏感数据保护 | 凭据哈希不返回，导出脱敏，环境密钥不暴露 |
| NFR-10 | 并发目标 | 500 用户同时登录，成功率 >= 99.7% |

## 6. 数据来源

### 广西本地观测数据
- 来源：`scripts/fetch_guangxi_data.py` 脚本
- 站点：`GX-HZ-001`（广西惠州鲈鱼养殖基地）
- 参数：溶氧 5.1-7.8 mg/L，pH 7.0-7.7
- 周期：24 小时逐小时采集
- 标识：`source_mode=crawled`
- 采集方式：尝试从广西公共数据开放平台、生态环境部数据中心等公开 API 采集；降级时基于广西生态环境厅公开参数范围生成

### 模拟演示数据
- 标识：`source_mode=simulation` 或 `execution_mode=simulation`
- 范围：低溶氧注入、告警/建议、增氧执行、通知送达、密度分析、效益对比、归档销毁
- 声明：模拟记录仅证明系统流程和权限规则可运行，不证明真实塘口调控效果
