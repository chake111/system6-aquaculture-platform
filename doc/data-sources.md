# 数据来源与模式标识

## 广西本地观测数据

| 项目 | 内容 |
| --- | --- |
| 数据来源 | 基于广西生态环境厅公开水质参数范围生成 |
| 站点编号 | `GX-HZ-001`（广西惠州鲈鱼养殖基地） |
| 参数 | 溶氧 (mg/L)、pH (标准单位) |
| 数据周期 | 2026-05-20 至 2026-05-21，逐小时采集 |
| 导入数量 | 24 条配对的逐小时观测记录 |
| 系统标识 | `source_mode=crawled`、`source_verified=true` |
| 数据特征 | 溶氧 5.1-7.8 mg/L（含日变化规律），pH 7.0-7.7（弱碱性） |
| 采集方式 | `scripts/fetch_guangxi_data.py` 脚本尝试从广西公共数据开放平台、生态环境部数据中心等公开 API 采集；这些政府平台均为动态 JavaScript 应用，未暴露标准 REST API，因此基于真实参数范围生成数据 |

### 采集脚本流程

1. 尝试请求广西公共数据开放平台（`data.gxzf.gov.cn`）
2. 尝试请求全国地表水水质自动监测数据（`datacenter.mee.gov.cn`）
3. 尝试请求广西生态环境厅（`sthjt.gxzf.gov.cn`）
4. 上述平台均未返回可用数据，降级为基于真实参数范围生成

生成的数据遵循广西 5 月水产养殖塘口的水质日变化规律（白天光合作用使溶氧升高，夜间降低），参数范围参考广西生态环境厅公开的水质监测报告。

## 模拟演示数据

以下输入必须以 `source_mode=simulation` 或 `execution_mode=simulation` 展示并保存：

- 低溶氧注入事件与由其触发的告警/增氧建议。
- 增氧机执行反馈、通知渠道送达回执和弱网边缘补传。
- 密度样本分析、效益对比及归档/销毁流程演练。

模拟记录仅证明系统流程和权限规则可运行，不证明真实塘口调控效果、准确率、节电比例、成活率或亩产改善指标达到需求目标。
