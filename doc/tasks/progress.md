## 依赖检查

- [x] 已确认演示使用模拟适配器或真实设备范围，并将 `source_mode` / `execution_mode` 策略固定到实现与测试中（`TBD-01`）。**默认决定：使用可替换的模拟适配器，所有模拟读数明确记录 `source_mode=simulation`。**
- [x] 已确认或模拟短信、语音、钉钉、气象服务方案，并为失败/重试准备测试场景（`TBD-02`）。**默认决定：定义 provider 接口并实现确定性的模拟 provider，覆盖成功、失败、超时和重试测试。**
- [x] 已确认真实阈值、生长参数和效益口径；未确认前仅使用版本化演示规则并标注用途（`TBD-03`）。**默认决定：使用版本化的演示规则/fixture，展示规则版本和"演示/未验证"标志。**
- [x] 已完成离线授权与数据保留安全评审，明确 7 天授权/缓存边界和敏感操作限制（`TBD-04`）。**默认决定：离线授权最长 7 天，仅支持只读缓存/处置草稿；禁止离线执行敏感操作。**
- [x] 已确认以 500 并发作为验收基准，或记录升级到 1000 并发后的测试资源和判据变更（`TBD-05`）。**默认决定：以 500 用户同时登录且成功率不低于 99.7% 作为验收基线。**
